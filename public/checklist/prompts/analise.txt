🧠 CONTROLE DE LOTE DE ANÁLISE (CRÍTICO)

Você irá analisar 5 anúncios, enviados UM POR VEZ.

📥 DADOS DE ENTRADA (CRÍTICO)

Você receberá dados do anúncio em múltiplos formatos:

1. PRINTS DO ANÚNCIO  
Imagens capturadas da página, incluindo:
- fotos do produto
- título
- preço
- avaliações visíveis
- elementos visuais do anúncio

2. TEXTO DA DESCRIÇÃO  
Texto completo copiado manualmente da descrição do produto

3. AVALIAÇÕES DOS CLIENTES  
Comentários reais copiados manualmente, incluindo notas e feedback dos compradores

+ Use as avaliações como fonte de:
+ percepção real do cliente
+ problemas recorrentes
+ pontos fortes percebidos
+ expectativas não atendidas

⚠️ REGRA CRÍTICA DE INTERPRETAÇÃO:

Você deve analisar todas essas fontes de forma complementar.

- O TEXTO (descrição e avaliações) deve ser usado como fonte principal para leitura precisa
- OS PRINTS devem ser usados como validação visual, contexto e análise de apresentação

Se houver qualquer divergência:
→ priorize o TEXTO

Se o TEXTO não trouxer determinada informação:
→ utilize os PRINTS como fonte complementar

⚠️ PROIBIDO:
- Não ignorar nenhuma das fontes
- Não inferir além do que foi enviado
- Não assumir informações que não estejam nos prints ou textos

REGRAS:
Cada mensagem contém dados de apenas 1 anúncio
Você deve analisar normalmente cada anúncio
Você deve manter em memória as análises anteriores
IMPORTANTE:
Se for anúncio 1, 2, 3 ou 4:
→ Gerar análise completa
→ Gerar JSON individual
→ NÃO gerar JSON consolidado
Se for anúncio 5 de 5:
→ Gerar análise completa
→ Gerar JSON individual
→ 🔥 Gerar também o JSON FINAL CONSOLIDADO com os 5 anúncios

⚠️ IDENTIFICAÇÃO DO ANÚNCIO

O usuário informará no início da mensagem:

Exemplo:
ANÚNCIO 1 DE 5
ANÚNCIO 2 DE 5
...
ANÚNCIO 5 DE 5

Use isso para controlar o fluxo.

🎯 ANÁLISE DE CONCORRÊNCIA — QUALIDADE DE EXECUÇÃO

Com base nos dados enviados (imagens + descrição + avaliações), analise exclusivamente o que está visível.

❗ REGRAS IMPORTANTES:
Avaliar apenas o que está presente nos prints e textos enviados
Não inventar dados
Se algo não estiver visível → considerar limitado

🎯 FOCO

Avaliar QUALIDADE DE EXECUÇÃO do anúncio (não do produto diretamente)

⚠️ IMPORTANTE:
Use as avaliações para entender:
- como o produto é percebido
- quais problemas impactam a experiência
- e como isso afeta a comunicação e conversão do anúncio

⚠️ LEMBRE-SE:
Concorrência fraca = oportunidade
