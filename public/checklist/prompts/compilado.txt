⚠️ REGRA CENTRAL DE INTERPRETAÇÃO (CRÍTICA)

Você deve entender corretamente o papel de cada entrada:

JSON DO PROMPT 1 = VISÃO MACRO DO MERCADO

Esse JSON mostra a realidade geral da página de busca, ou seja, o retrato mais amplo do mercado visível.

Ele é a principal fonte para entender:

o nível real de concorrência no mercado
a proporção de anúncios fortes e fracos
a presença de atributos competitivos relevantes
a distribuição visível de volume
o padrão dominante do mercado
JSON DO PROMPT 2 = VISÃO MICRO / LUPA QUALITATIVA

Esse JSON mostra uma amostra intencional de 5 anúncios, escolhidos para enxergar o mercado com mais profundidade.

Esses 5 anúncios NÃO representam estatisticamente o mercado inteiro.

Eles servem para:

validar qualitativamente os padrões do mercado
mostrar com mais profundidade como os concorrentes executam
identificar falhas recorrentes
revelar oportunidades reais de posicionamento, comunicação, oferta e conversão

⚠️ IMPORTANTE:
Os 5 anúncios foram escolhidos com propósitos diferentes (bom, ruim, líder, diferenciado, problemático).
Portanto, NÃO use esses 5 anúncios como se fossem a distribuição do mercado.

Exemplo do que NÃO fazer:

ver 1 anúncio forte e 1 fraco e concluir que o mercado é “50% forte / 50% fraco”

Isso seria errado.

A distribuição real do mercado vem do Prompt 1.
Os 5 anúncios servem como amostra qualitativa, não como distribuição quantitativa.

⚠️ HIERARQUIA DE PESO ENTRE AS FONTES

Ao montar o JSON final, use esta lógica:

Quando a análise for sobre o MERCADO COMO UM TODO:

→ dê mais peso ao JSON DO PROMPT 1

Quando a análise for sobre PADRÕES DE EXECUÇÃO / FALHAS / OPORTUNIDADES:

→ use o JSON DO PROMPT 2 como lupa qualitativa

Quando um critério depender das duas visões:

→ use:

Prompt 1 como base principal
Prompt 2 como validação qualitativa / aprofundamento

📊 CRITÉRIO 2: CONCORRÊNCIA

═══════════════════════════════════════════════════════════

📊 CONCORRÊNCIA (USAR MACRO + MICRO)

═══════════════════════════════════════════════════════════

A nota de CONCORRÊNCIA deve ser construída a partir de duas camadas:

CAMADA 1 — BASE PRINCIPAL = JSON DO PROMPT 1

Esse JSON mostra:

quantos anúncios parecem fortes
quantos parecem fracos
quantos são mais vendidos
quantos são diferenciados
a proporção visível de atributos competitivos:
FULL
frete grátis
entrega rápida

Essa é a base principal da nota, porque representa o panorama real do mercado.

CAMADA 2 — VALIDAÇÃO QUALITATIVA = JSON DO PROMPT 2

Esse JSON mostra:

como os concorrentes fortes executam
quais são as falhas dos fracos
quais padrões se repetem
onde há espaço real para entrar melhor
LÓGICA OBRIGATÓRIA:
Use a distribuição (%) do Prompt 1 como peso principal
Use o Prompt 2 para qualificar essa leitura
NÃO use média simples dos 5 anúncios
NÃO ignore a distribuição do mercado
A nota final deve refletir a realidade do mercado, não apenas os exemplos escolhidos
INTERPRETAÇÃO:
maioria fraca → nota de concorrência mais baixa
maioria média → nota intermediária
presença relevante de fortes → elevar nota
presença forte de FULL / frete grátis / entrega rápida → elevar dificuldade competitiva
mercado com muitos atributos logísticos fortes e boa execução → concorrência mais dura
mercado fraco em execução, mal posicionado e com poucos diferenciais → concorrência mais acessível
IMPORTANTE SOBRE ATRIBUTOS LOGÍSTICOS:

FULL, frete grátis e entrega rápida não são apenas logística — eles também aumentam a força competitiva do mercado.

Exemplo:

se muitos concorrentes são FULL
se muitos oferecem frete grátis
se entrega rápida aparece com frequência

isso melhora a proposta percebida pelo comprador e aumenta a dificuldade de competir, especialmente para quem ainda não terá FULL.

Portanto, esses atributos devem influenciar:

a nota de concorrência
o parecer sobre posicionamento
a leitura do padrão atual do mercado
NO CAMPO "parecer":

Mencione explicitamente:

a leitura do mercado com base na distribuição do Prompt 1
como os 5 anúncios analisados confirmam ou aprofundam essa leitura
o impacto dos atributos competitivos visíveis (FULL, frete grátis, entrega rápida)
NO CAMPO "oportunidades":

Liste oportunidades reais encontradas com base:
